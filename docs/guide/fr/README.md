# Q‑Art — Guide d'utilisation

> **Propriétaire** · © 2026 Ideotion · usage personnel et non commercial autorisé ; tous les autres droits réservés (`LICENSE`).
> 🇬🇧 **English version: [`../`](../README.md)**

Q‑Art vous aide à vous débloquer face à une décision difficile — non pas en vous donnant une réponse, mais en promenant votre question sur toutes les facettes de la situation jusqu'à ce qu'une **meilleure question** apparaisse. La réponse vit le plus souvent à l'intérieur.

| À lire | Quand |
|---|---|
| **[Démarrer](./demarrer.md)** | Vos dix premières minutes — de l'installation à la première synthèse. |
| **[Le parcours complet](./parcours.md)** | Toute la méthode sur un exemple réel, avec captures d'écran. Si vous ne lisez qu'un document, c'est celui-là. |
| **[Choisir votre entrée](./choisir-une-entree.md)** | Atlas, Socrate ou Cartes — ce qui vous convient aujourd'hui. |
| **[Vos données](./vos-donnees.md)** | Où vivent vos mots, comment les sauvegarder, comment tout effacer. |
| **[FAQ](./faq.md)** | Est-ce une thérapie ? Hors ligne ? Où sont mes données ? |

**Une promesse, d'entrée :** tout ce que vous écrivez reste sur votre appareil, chiffré. Pas de compte, pas de serveur, pas de pistage. ([Détails](./vos-donnees.md).)

> Q‑Art vous aide à réfléchir. Ce n'est ni une thérapie, ni un avis médical, juridique ou financier. En cas de détresse, contactez votre numéro d'urgence local (112 UE · 911 US · 999 UK) ou une ligne d'écoute — la page **À propos** de l'application porte aussi cet avis.
